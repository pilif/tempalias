tempalias provides temporary, anonymous email addresses

Ever wanted to sign up for any kind of online service or community without providing your real email address, for example because you don't want to get spammed? tempalias helps you out by providing temporary, anonymous email addresses, also called "aliases". Any message sent to the tempalias address will be forwarded anonymously to your real email address. You can set a maximum number of days or messages for which the tempalias will be valid. When either limit has been reached, your temporary address will automatically be deleted (Read privacy for details).

No registration, no cost and no ads

Unlike other, similiar services out there, tempalias doesn't require you to create an account or provide any personal information (other than your real email address, of course). Also, the service is completely cost and advertisment free.

Background and tech stuff

tempalias is a grown-up proof of concept showing that it is possible to easily write both a web server and an email server in JavaScript using node.js. It has initially been written by Philip Hofstetter (@pilif) based on an idea from the year 2002. All the public facing design you see has been done by Richard Haydon. Professionally, Richard is a photographer, so if you are looking for one, consider his services.

If you want to see how tempalias is done or if you want to run a comparable service yourself, the tempalias.com codebase is avilable licensed under the MIT license at http://github.com/pilif/tempalias

If you want to learn how Philip went from "no idea about asynchronous programming with node" to "completed tempalias.com", have a look at a series of blogposts about the topic.
